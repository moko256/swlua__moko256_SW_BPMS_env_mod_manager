This is an addon managing env mods for Stormworks.


Features


    
    
  • Spawn/despawn env mods
    • 
    
  • Spawn/despawn addons whose name starts "SW_BPMS_" and that have locations
    • 
    
  • Support using in either single-player mode or multi-player mode
    • 
    
  • Display 'Display Name' properties on map
    • 




How to use


    
    
  1. Subscribe this addon.
    2. 
    
  2. Open the new world screen and select this addon and select other env mods what you want to use with.
    3. 
    
  3. Create the world. Only these steps and you can use those commands. Have fun!
    4. 




Command usage



    
CommandPermissionDescription

    
?bm lAutheddisplay all addons managing

    
?bm s aAdminspawn all

    
?bm s [num]Adminspawn at num

    
?bm d aAdmindespawn all

    
?bm d [num]Admindespawn at num

    
?bm hAutheddisplay this help



[num] is the number in [] in the output of '?bm l'.


Note


    
    
  • This addon cannot add any other new addons/env mods to world. You have to select them when you are creating world.
    • 
    
  • This addon is like prerequisite mod. This addon cannot use alone.
    • 




For env mods creators: labels and icons


    
    
  • Vehicles/objects' "Display Name" is dealt by this and will be used as map labels' text.
    • 
    
  • Vehicles/objects' "SW_BPMS_map_icon=[LABEL_TYPE]" tag is dealt by this and will be used as map labels. (ex: SW_BPMS_map_icon=13 will be fuel icon)
    •
