これはStormworksで環境modを管理するアドオンです。


機能


    
    
  • 環境modをスポーン/デスポーンさせる
    • 
    
  • 名前が"SW_BPMS_"で始まり、ロケーションを持つアドオンをスポーン/デスポーンさせる
    • 
    
  • マルチプレイで利用可
    • 
    
  • 'Display Name'項目をマップ上に表示
    • 




使い方


    
    
  1. このアドオンをサブスクライブする。
    2. 
    
  2. ワールド作成画面を開いてこのアドオンと、一緒に使いたい環境modを選択する。
    3. 
    
  3. ワールドを作成する。これだけで下のコマンドを利用できます。
    4. 




コマンド使用方法



    
コマンド権限説明

    
?bm lワークベンチ利用者管理しているアドオンを表示

    
?bm s a管理者全部スポーンさせる

    
?bm s [num]管理者numを用いてスポーンさせる

    
?bm d a管理者全部デスポーンさせる

    
?bm d [num]管理者numを用いてデスポーンさせる

    
?bm hワークベンチ利用者英語版ヘルプを表示



[num]は'?bm l'の出力の[]内の数字を利用します。


注意事項


    
    
  • このアドオンは新しく他のアドオン/環境modを追加できません。ワールド作成時に選択する必要があります。
    • 
    
  • このアドオンは前提modのように動作します。このアドオンは単体動作しません。
    • 




環境mod製作者向け: ラベルとアイコン


    
    
  • ビークル/オブジェクトの "Display Name" が処理されて地図上のラベルの文字列に使用されます。
    • 
    
  • ビークル/オブジェクトの "SW_BPMS_map_icon=[LABEL_TYPE]" タグが処理されて地図上のラベルのアイコンに使用されます。 (例: SW_BPMS_map_icon=13 で燃料給油所のアイコンになります)
    •
